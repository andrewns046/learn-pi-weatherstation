# Learn to Build a Raspberry Pi Weather Station !
Part List:
- DHT11 (Temperature & Humidity Sensor)
- 2 Buttons
- 4 LEDs
- 4 43 Ohm resistors
- 2 10k Ohm resistors
- 1 4.7k Ohm resistor
- Jumper Wires
- Breadboard
- Raspberry Pi
